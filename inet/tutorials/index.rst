Tutorials
=========

The following tutorials explain via step-by-step examples how to use various features
of the INET Framework. Each tutorial contains a series of simulation models of
increasing complexity.

You can browse the turorials pages on this web site. The source code of
the simulations (NED, ini and other files) and the web pages are in the
`tutorials/` subdirectory of the
`INET repository <https://github.com/inet-framework/inet/tree/master/tutorials>`_.
